MGMS - Technical Documentation

The technical documents provide the detailed design specifications for the various modules of Multisite Grant Management System project.

They do not contain the project plans nor do they detail the source code. It is a guide for system architecture and development, not for phasing, timelines or deliverables.

Finally, these are ‘living documents’ and the specifications will change, continuously, as the project proceeds. More details will be added and the existing information and edited as the requirements, architecture and design evolve in the course of the project. 

Summary

MGMS is built up on Groovy on Grails frame work. MySql data base is used as backend tool and for graphical dispaly Google Chart is used.
