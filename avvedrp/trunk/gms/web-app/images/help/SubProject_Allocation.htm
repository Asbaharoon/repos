Amrita - Multi Site Grant Management System

   

Sub Project Allotment 

 

 

The project is divided into subprojects and the fund allocated for that project is allocated among its subprojects.

This can be done only after the account headwise allocation.

The fund along with the subproject is allocated to the recepient which is selected.

The date of allocation & sanction order number need to be entered.

The fund allocation details will be listed in the list given below.

The status of the subproject can be set by clicking Grant Withdrawal/Closure Link.
