Amrita - Multi Site Grant Management System

Assign Projects to PI

 

Select the Investigator

Select the Project to be mapped

Select the role(PI or Co-PI) of the Investigator of that particular project

Click the create button, now the Project is been assigned to the Investigator

Any number of Projects can be assigned for a single Investigator

Please Note that an Investigator who is playing the role of PI in one project can also act as Co-PI in some other projects
