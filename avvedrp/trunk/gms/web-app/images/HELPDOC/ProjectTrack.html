Project Status & Closure

Using this functionality the institution can record the status of its project, at various points of project duration, finally setting the status “Close” when the project gets over.

After logging in to MGMS, in the Project List page, click on the name of the particular project. The project home is opened with default page showing the Project Info. Click on top row menu Project Closure.

The Project Closure Entry page is opened

  • The project status and the percentage of completion will be saved against the date of the particular tracking entry and so also, any remarks by the user.
  • The entries may be edited later on.

On saving project status as “closed” for a project, the Project List page is displayed to the user with the message informing that the project has been closed successfully. After closing, all users have access to only the Project info page of the particuar project. (All other tabs are disabled.)
