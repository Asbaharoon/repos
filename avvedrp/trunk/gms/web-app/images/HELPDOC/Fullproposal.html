Full proposal submission (Internal)

Initial Administrative Functions

Adding new users - reviewers:

A new role “Reviewer” has been added to GMS. The reviewers can be added by the Administrator through Add User page and the role can also be assigned here.The reviewers created are displayed in the User List.

The privileges for the role “reviewer can be saved in the Access Permission page.

Creating approval Authorities:

On the left menu panel, the administrator clicks on Approval Authority menu. It expands to display the sub menu – Add Approval Authority
On clicking Add Approval Authority, the page opens up where each group of approving persons at the institution is registered. Further, for each approval authority, the members are also added.

The fields bordered in red are mandatory.
The personal details are: Name of the group, group e-mail ID and Institution. Further, the Approve Mandatory field decides if the particular group’s approval is mandatory and it is set to “Y” (Yes) by default. View All is set to “Y” (Yes) by default which indicates that the members of this group are privileged to view all proposals even if they don’t get to review them. Setting a group inactive here will result in removing the group from all user options. It is then deemed to be non-existing.
In the Approval Authority List, for each line item (approval authority) , clicking on Add Members opens the page below, where members can be added to/removed from each Approval authority. The existing reviewers are displayed in the table on the right from which selection can be made.

In the Approval Authority List , for each line item (approval authority), clicking on Edit, opens the page below, where any modification can be made.

Submission of Full proposals by staff members

The approved pre-proposals details also get automatically populated in the Full Proposal List for the Staff.

By clicking Submit Full proposal for a line item here, the staff will open the proposal uploading page. The Full Proposal Status displays No Full Proposal for both the line items indicating that proposals have not been uploaded yet. By clicking Submit Full Proposal for each line item, screen can be opened where the proposal can be uploaded and saved or submitted (just as in the case of pre-proposal)

Once the proposals are submitted, the status in Full Proposal List – Full Proposal Status column changes to Reviewing.

Here onwards the process is same as that for pre-proposal. The admin maps the approval authorities for the submitted full proposals and so the details get populated as lie items for the various reviewers.

The approved full proposals are ready to be sent to external agencies and this submission is done by a privileged user.
