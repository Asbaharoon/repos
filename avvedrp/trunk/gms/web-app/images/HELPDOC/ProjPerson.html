Project Personnel

In this section, there is provision to add personnel against each project. 

After logging in to MGMS, in the Project List page, click on the name of the particular project. The project home is opened with default page showing the Project Info. Click on top row menu Project Personnel.


The details saved are:

    • Details such employee name, employee no:, DOB,  joining date, relieving date from the project) , designation etc.
    • Qualification details such as University, exam passed, % marks, passing out year etc.
    • Experience Details such previous employer’s name, joining/relieving dates and designation while there.
    • Salary Details such as salary component, corresponding amount , date with effect from etc

    Clicking on Edit for a particular employee in the table will open the Edit Project Employee screen. Here the employee details can be edited.
