Proposal Evaluation - Eligibility Check

The Notification List page lists out all the projects for which proposals have been requested by the user’s institution (grantor). From here, the grantor can do the following, for each project:

  • View the application form
  • Upload attachments
  • Publish notification
  • View proposals already submitted by interested institutions
  • Edit notification

When the notification is published, all institutions in the system will get the notification in their Proposal Management section of the GMS. 
Proposals are uploaded against these notifications and can be viewed and evaluated by the grantor to first have a preliminary elimination round based on prequalification (eligibility criteria)

After logging in, click and expand the Project menu in the left navigation bar. Then click on sub menu Call for Proposal. The Notification List page opens. This section lists out all the notifications published by the user's institutions in the GMS and indicates if for each line item (notification) any proposals that have been received. .

Clicking on View in Proposal column of Notification List for a specific notification, the following Proposal List screen opens up – all the proposals in response to the specific notification are listed here.

Clicking on View for a line item will show the summary of the proposal details on Show Proposal screen. Clicking on View on this screen, opens the application form (non editable).

Returning to the Proposal List screen …..Click Review link in Eligibility Check column for the proposal that was opened above.

The criteria populated here were originally saved through the Settings module. The user can enter his evaluation and observations

On clicking Submit, there is a status message, the Proposal Status changes to that which is set in the previous screen (above). The eligibility check status becomes “Submitted”.

The criteria for eligibility check are saved through the Settings module by the site admin. Currently, the same set of criteria will be displayed for every project’s proposal evaluation.
