External proposal submission in response to private notification

Using this feature, an institute can submit proposals in response to notifications published privately by any other institute in the GMS network.

After logging in, click and expand the Project menu in the left navigation bar. Then click on sub menu Proposal Management . The Proposal List page opens. This section lists out all the notifications published by the other institutions in the GMS.

On this page, the interested institution users (prospective grantees) can view

  • Project name - the title of the grant opporunity according to the granting agency
  • Notification date - the date on which the granting agency published the notification within the GMS thereby the item appearing in all other user institution's Proposal List.
  • Grantor institution’s name
  • Last date for submission of proposal - after which, proposal submission will be impossible as the button "Apply" will be disabled.
  • Notification details - View and download all files associated with the notification details
  • Status - status of submission (for the particular user's institution) - "Submitted" or "Not submitted"
  • Apply - Open and fill up application form required by the Granting agency.

Once the details are filled in the form, the user can "Save" draft without submitting it or submit it directly. In both cases, a Controller ID is generated and sent automatically to the user's email id. In case of "Save"d drafts, this controller id is requested each time the form needs to be opened. Once submitted, no further modification is possible.
