Notification

For each grant scheme or any other funding opportunity , the grantor institution can publish the notification requesting for proposals along with the application form. The Site Administrator and the Proposal Administrator have the privilege to create and publish notifications.

The user clicks on Projects tab, to expand the menu and then clicks Call For Proposal. Here, clicking on top row menu - New Notification, will open the page where new notification can be created.

The functionality provided allows the institution user to create a notification on a wordpad and save alongside, the associated notification name (grant), notification code, notification date and last date for proposal submission.
The related application form can be uploaded here.

The Notification List page lists out all the projects for which proposals have been requested by the user’s institution (grantor). From here, the grantor can do the following, for each project:

  • View the application form
  • Upload attachments
  • Publish notification
  • View proposals already submitted by interested institutions
  • Edit notification

There is a special flag "Public YesNo" which has been set "Y" by default for any notification created in GMS. This indicates that the notification. along with the application form and other attachements are to be made available outside the GMS application, for the benefit of the common public. Anybody (not necessarily a user of GMS) may apply to the notification without logging in.

In the GMS Log-in page, clicking on link Upload Proposal will display the Notification List for all institutions which are part of that GMS network Clicking on button Apply for any notification will open the correponding application form.

In case the flag has been set "N" then the particular notification is for private access and application only. Only GMS users of that network can view the notification in the internal Notification List - under tab Projects, sub menu - Proposal Management.

 

Proposal Management

For each institution's privileged user, this section lists out the (request-for-proposa)l notifications published by all the institutions in the GMS for various projects.

On this page, the interested institution users (prospective grantees) can:

  • View the notification title, code, notification date, grantor institution’s name, last date for submission of proposal etc.
  • View the notification details
  • Upload their proposal thereby setting a status “Submitted” and then
  • Apply for the grant scheme..
