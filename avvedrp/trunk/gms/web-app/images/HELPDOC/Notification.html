Notification

For each grant scheme or any other funding opportunity , the grantor institution can publish the notification requesting for proposals along with the application form.
The functionality provided allows the institution user to create a notification on a wordpad and save alongside, the associated project, notification code, notification date and last date for proposal submission.
The related application form can be uploaded here.

The Notification List page lists out all the projects for which proposals have been requested by the user’s institution (grantor). From here, the grantor can do the following, for each project:

  • View the application form
  • Upload attachments
  • Publish notification
  • View proposals already submitted by interested institutions
  • Edit notification

When the notification is published, all institutions in the system will get the notification in their Proposal Management section of the GMS. 

Proposal Management

This section lists out the request-for-proposal notifications published by all the institutions in the GMS for various projects.

On this page, the interested institution users (prospective grantees) can:

  • View the project name, notification date, grantor institution’s name, last date for submission of proposal etc.
  • View the notification details
  • Upload their proposal thereby setting a status “Submitted” and then
  • Apply for the project.
