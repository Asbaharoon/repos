Change Password

The Change Password option is given to all users of MGMS.

To access this feature, after logging in to the application, on the left navigation bar, clickChange Password. This page can also be accessed by clicking on the yellow key icon in the top banner . In the resulting screen, enter the old password, user's choice of new password and then confirm it by retyping it in a different field.

The current rules to be satisfied while setting a new password are:

  • Password should be 6 to 12 characters long.
  • Password should have at least one alphabet.
  • Password should have at least one numeric value
  • Password should not have any special character. For e.g. @ - # _ etc

Now click button Change Password. For logging into MGMS the next time, the user will have to use the new password.
