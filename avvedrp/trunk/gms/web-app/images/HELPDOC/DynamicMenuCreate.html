Dynamic Menu

Features

On this page, the admin can decide on the pages to be made available for users with different roles in MGMS.

Currenty, this feature is given to only the super administrator role.

Click menu User Management (on the left) and select sub menu Add Menu.

In the Add Menu page, select the role from the drop down list.

Select the pages to be provided from the Unassigned Menu List and click the right arrow. The selected page names are now displayed in the Assigned Menu List on the right. (A selected page can be removed from this list by clicking the left arrow and returning it to the Unassigned Menu List).

Clicking the Create button will enable the users with that role to access the particular pages in GMS and perform the corresponding activities.

The privileges set here will be applicable to all those users belonging to the institutions which are members in this particular hosted GMS
