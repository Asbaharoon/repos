Award Project

Once all the received proposals are evaluated by the approval authority, the best of the lot are awarded projects i.e the grantor institute officially confirms that it will fund the projects as described in the winning proposals. This is applicable for each grant scheme notification issued by an institute.

Following are the salient features:

  1. Projects are awarded to own or other institutions’ PI (as mentioned in the proposal).
  1. It is not necessary that the entire proposed project be accepted and so the project title may vary from the proposal title.
  2. Immediate essential details captured as discrete data are– sanction order no: (a unique official identifier) project start/end date, total amount allocated.
  1. A lot more details of the contract can be captured. Currently, provision for attachment upload has been given. (Further details are then captured such as Terms of the project, Fund allocation, Standard Expense Heads, Frequency of fund Release, formats for progress reports, formats for financial Reports, their frequency etc.) Document type is Project and Attachment type – contract.
  1. Multiple projects can be awarded for a single notification.
  1. Projects maybe awarded at different times over a period i.e. all winning projects are not selected and awarded in a single session. (Even as a project is happening, new proposals will be under evaluation, awarded etc).

Steps:

In GMS, currently it is possible for the proposal administrator and site administrator to view the list of reviewed proposals for each notification. Final score after evaluation by the review panel is displayed for each proposal.

This is in the Call for Proposal sub menu of Projects menu - on opening which,

An additional column will appear for  “Award” link  beside “Proposal Review” link  (for site admin & proposal admin), immediately after the relevant reviewers complete and submit evaluation for a proposal of a notification.

On clicking “Award” for an evaluated proposal , a new page opens where the proposal title, project title, institution and PI are automatically populated. The project title and PI details can be editable (in case of a scenario where PI/Project nature is changed. For e.g. if only a component of the proposal is awarded a grant.)

In the Proposal List page, “Award” link is now replaced by “Awarded”. On clicking “Awarded” the details can be viewed.

The first time a proposal is converted to project in the above manner, the notification is converted to a project and displayed in the Project List for that institution (which published the notification). The awarded project is now listed out as a sub project for this project.

The awarded project will also get automatically displayed in the Project List of the awardee (grantee).
