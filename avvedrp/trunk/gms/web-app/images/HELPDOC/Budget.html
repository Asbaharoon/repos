Budget Creation and mapping

In Budget Module, the site admin (currently only site admin has privilege) can create budgets for financial years. These budgets should have amount allocations to different account heads. The created budget may be assigned to one or more full proposals, created projects, sub projects etc.  

Click Budget tab from the list of tabs on the left. The menu expands to display sub menus - Financial Period, Budget Master & Assign Budget to Module.

Financial Period

Financial periods are setup firs for easy categorizing and assigning of created budgets.

Budget Master

In next step, Budget Master is created for the selected financial period. Budget is created by saving details - budget title, code, detailed description and total amount.

Budget Allocate

For any budget created, the total budgeted amount maybe split up amongst several account heads. This details budget can be assigned to any proposal, project, notification etc.

If for a budget master, "Edit" button and “Budget allocate” are clicked for an already assigned budget, a pop up message will appear -"This budget is already assigned. Any changes made now will modify the assigned budget also. Do you wish to continue?"

Finally, you can map the created budget master with a Proposal, Project or Notification title.

Assign Budget to module

In this screen, a selected budget can be assigned to a specific project or notification or full proposal etc by the site admin.
On selecting the module type, all active titles for that module type will be listed in the module dropdown.
Please note that for a selected module type, only active titles will be made available for budget to be assigned.

  • For module type Notifications – only those which have not been published
  • For module type Full proposals– which have not been submitted  
  • For module type Projects – only own Projects /sub projects which have not ended

Click “Create” to assign the budget to a selected module.

Viewing assigned budgets

For notifications - "view" option is provided in the Notifications List for each line item – to display assigned budget.
For full proposal - to be viewed by reviewers also - "view" option is provided in the Proposals List- to display assigned budget.
For projects, a sub menu is provided at the top (under proj info)- "Assigned budget" where budget assigned to the project will be displayed.
