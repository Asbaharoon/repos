Fund Advance

Advances may be given to the project employees from the project fund to cover possible future expenses – for e.g travel advance to attend a conference. Currently, it is not possible record this advance but only any expenses paid for in the various account heads.

In GMS Fund Advacne module, you can record the details of any advance paid for the Project.

Since advance fund is against grant allocation the total expense and advance should be less than or equal to allocated amount (i.e.  allocatedAmount >=(totalExpense+FundAdvance))

From menu Expenses, select sub menu Fund Advance

Mandatory details to be filled in are - Advance date & amount, associated grant allocation and a short description, payment mode and payment details (check and bank details etc).

Click Create to save the details

In the table Advance Fund List below, the saved details are displayed in a line item. All such advances made for this project are displayed here as line items/rows.

Each row has two buttons - Edit and Enter Expense Details

  • Edit - Any Advance information maybe updated here at a later time. But please note that the advance details are not editable once the first expense is entered against that advance.
  • Enter Expense Details - In the new screen, each expense detail can be entered for which payment came from the particular advance. You can enter multiple expenses at different times but only till the advanced amount lasts. An expense for which the remaining fund is not sufficient will not be allowed to be recorded.

Please note that in the Record Expenses screen (under Expenses menu), Current Balance amount displayed has all such advances deducted since those amounts are not available for utilizing.

Closing an Advanced Fund

In the Advance Fund List (Fund Advance screen), each line item has an option “Close” – a link provided in the Advance Closure column. On clicking the “Close” link, a new page opens.

On clicking Close Advance, message appears - "Are you sure to close this advance and refund the remaining advance?"

Click "OK" if there are no more expenses to be recorded and the remaining fund is to returned to the project. Message appears stating that the advance is closed successfully (and the link now appears "Closed")

On clicking the link, the Closure details appear displaying the refunded amount as well.
