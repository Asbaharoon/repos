External Funding

Providing funds to other institutions

An Institution can provide fund for a project of any other institution within its MGMS network.

Select ‘External Fund’ sub menu under Institution main menu on the left. This link will open the External Fund page.

On this page, the external fund allocation details are captured first. Then, corresponding to each allocation, the multiple transfers made to the recepient are recorded. In case of refundable loans, the refund details are also captured.

First select the intended recepient Institution from the dropdown list and then select the project at that institution for which the funding is to be given.

Grantor workflow:

External Fund Allocation

Enter details of the funding for e.g sanction order no:, date of sanction, amount value, type of fund (refundable/non-refundable) etc.

Click Create

The details appear in a line item in the External Fund List below.

Transfer Fund link provided for each line item in this list will open the Fund Transfer page. You can save details of multiple transfers made according to this allocation.

Refund list link will lead to a screen where all refunds of grantee against that grant allocation will be listed. Please note that this link is visible only if the type of allocation is "Refundable".

Grantee workflow

Select the project from the Project List

From top row menu Allocation, select sub menu External Fund.

External Fund Allocation List is displayed.

Each line item corresponds to an allocation recorded by the respective grantor. And you may capture details of the funds received, expenses paid for as well as refunds against each of these allocations.

The link Receive Fund will open the page, where you can receive the funds transferred by the grantor.

The link View Expenses will open the list of expenses for which payment was made from this fund.

The link Refund will open the page where any refunds back to the grantor can be recorded. These details can be viewed by the grantor.

Receiving funds from external institutions

An institution may receive funds from external sources for a specific project, aside from the designated grants. These institutions may be funding agencies, angel investors, banks etc and moreover, may not be registered in your MGMS network.

The details of such received funds can be captured for each project in MGMS.

Adding Funding Agency details in MGMS

From menu Settings on the left, select sub menu Grant Agency.

Create Grant Agency page opens up.

Enter details of the organization - name, short unique code, address, contact number, email id etc

Click Create.

The newly added agency details now appear in the list of Grant Agencies displayed at the bottom of the Create Grant Agency page.

(Receiving) fund management

Select the receiving project from the Project List.

Select sub menu External Fund from the top row menu- Allocation

Select the External funding agency, and enter the details of the allocation. In case Type of fund is refundable then the option to save refund details are provided to the user.

Click Create.

Multiple allocations can be saved for each project.

For each line item in the External Fund list, links are provided to do tasks – edit allocation details, transferring funds and view refunds made.

Click link Transfer Fund to open the screen below and record the details. Multiple transfers can be made against the allocation till the allocated amount is fully transferred.

(Please note that the receiver records both the transfer and receipt of funds. )

To record receipt, select menu Expenses and then select sub menu Fund Receipt

Select the allocation and the transfer from the drop down lists. Record the other details of the receipt.

Click Create

Please note that only one receipt is possible for each transfer.

Refund

In the Received Fund list (in the Fund Receipt page), each line item has a link Refund Amount.

In the opened page, capture refund details and click Create. The refund details can be viewed by clicking link Refund List in Allocation menu, External Fund sub menu - External Fund List, for each line item.
