Sub project Allocation

Create sub project
The institution can divide the main project into sub projects to be further assigned to other institutions.

  • The details saved of a sub project are: name, start & end dates, receiving institution, amount of allocation, sanction order number (Letter no :), date of allocation and any remarks by the user.
  • Now the user’s institution has become the “Grantor” who has assigned a (sub) project and allocated an associated grant amount to the receiving institution, the Grantee.
  • What is categorized as “sub project” by the grantor is categorized as “project” by the grantee
  • Grant allocations may be made any number of times to a grantee, for a project.
  • The same project cannot be assigned to two/more different institutions.
  • (The system validates that sub-project start and end date falls within the start and end date of the main project).
  • After creating a subproject, any number of associated documents may be uploaded into the GMS, for future reference.
