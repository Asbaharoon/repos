Sub project Allocation

Create sub project
The institution can divide the main project into sub projects to be further assigned to self or other institutions. Ideally, only the main project PI, co-PI and the institution's administrator can create sub projects.

After logging in to MGMS, in the Project List page, click on the name of the particular project. The project home is opened with default page showing the Project Info. Click on top row menu Allocation and further,click on the sub menu Sub allocate project.

The Sub project Allotment screen is opened.

  • The details saved of a sub project are: name, start & end dates, receiving institution, amount of allocation, sanction order number (Letter no :), date of allocation and any remarks by the user.
  • Now the user’s institution has become the “Grantor” who has assigned a (sub) project and allocated an associated grant amount to the receiving institution, the Grantee.
  • What is categorized as “sub project” by the grantor is categorized as “project” by the grantee
  • Grant allocations may be made any number of times to a grantee, for a project.
  • The same project cannot be assigned to two/more different institutions.
  • (The system validates that sub-project start and end date falls within the start and end date of the main project).
  • After creating a subproject, any number of associated documents may be uploaded into the GMS, for future reference.

After creation of sub projects the grantee is connected to the sub project in the following manner throughout the life cycle of the sub project:

  • Edit sub project details such as PI , Amount allocation, start/end dates etc
  • Transfer funds to the sub project from time to time against the allocation
  • Upload Attachments relevant to the sub project
  • If required, record the withdrawal of grants to the specific sub project - this means that the particular sub project will no longer be of interest for the grantor and there fore will not be financed from then on.

In the Sub project List (bottom half of the allotment page) , each line item (sub project) has links to enable user to access the respective screens mentioned above.

The utiization reports submitted by the institutions undertaking the sub projects will be available in the Reports module and is therefore discussed in that section.
