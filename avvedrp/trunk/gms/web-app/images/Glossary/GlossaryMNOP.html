M

Masters
Masters is the module where the settings particular to an institution are saved. Through the Masters module, the user can define his/institution specific options for entities such as account heads, PIs, roles, project types, proposal category etc. Once these lists are created in the GMS, the particular institution user can select from these options, where ever required in MGMS.

 

N

Notification
Notification is a solicitation or an announcement by the grantor to invite potential grantees for proposals for funding in specific topic or research areas.

Notification Code
In MGMS, Notification code is intended to uniquely identify each notification published by an institution throughout the grant cycle. Ideally, the reference system followed in the records of the particular institution can be implemented here.

 

O

Organization
Organization is a generic term used to refer to an educational institution or other entity, including an individual, which sponsors grant as well as applies for or receives a grant.

 

 

P

Percentage of completion
For a project, this number indicates the tasks already completed. By recording this detail over the project lifecycle, project status tracking can be done.

Permissions
Permissions denote the various access rights administered to particular role/group of users. The permissions include those for actions such as View, Edit, Delete at screen level as well as at field level within the application. For e.g. only site administrators are permitted to create new users for the institution.

PI
The PI, or Principal Investigator, is the individual named in the MGMS designated by the grantor or grantee as the one who is primarily responsible for carrying out the project.

Pre-proposal
Pre proposal is a short version of a proposal sent to own department/supervisor for internal review. It is used to determine the applicant's eligibility and how well the project can compete with other applications from the same institution and eliminate proposals for which there is little or no chance for funding.

Privilege
Privilege is a user right assigned by the site administrator to individual/group of users as part of the security settings.

Project
A temporary endeavor undertaken by an organization or individual to create a unique product, service, or result.

Project Closure
Project closure is the end of a grant programme. Ideally, it is marked by a meeting between the two parties i.e. the grantor & the grantee and it is ensured that necessary scientific, administrative, and financial reports as well as any pending/remaining funds have been received.

Project Code
In MGMS, Project code is intended to uniquely identify each created project throughout the application by a short familiar name. Currently the application would accept an arbitrary code which is familar to the user and display only the code in all relevant screens.

Project Status
Project tracking for its life cycle starting from "Open" to "Closed" is carried out by the privileged users by recording the date, status, percentage of completion and any significant remarks, at various times during the project.

Project Type
This is a user defined attribute to be assigned for each project created in the application. This will help categorize projects and enable easy search.  Examples - State, Central , PPP etc another set of project types can be web portal, infrastructure, database etc

Proposal Category
This is a user defined attribute to be assigned for each proposal created in the application. This will help categorize proposals and enable easy search.  Examples - State, Central , PPP etc another set of proposal types can be web portal, infrastructure, database etc

Proposal Type
This is a predefined attribute for any document that goes through the approval process in MGMS. Currently, the 2 types used are - Pre- proposal and Full proposal.

Publish Notification
This is the process by which proposals are requested for, from the user institutions of the MGMS enterprise. The proposal notification by an institution and which include details such as title/domain area, deadine for submission etc are displayed in the Notification Lists of all other institutions so that interested parties may apply.
