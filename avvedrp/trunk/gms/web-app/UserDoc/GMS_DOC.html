Amrita – Multi Site Grant Management System

 

Overview:-

 

Ministry of Human Resource has elaborate fund allocation to cover a variety of projects located in various places. Our grant management software has been designed to meet the unique needs within higher education and research institutes, including the specific needs of individual departments, centers, schools, colleges, and universities. The grant management software design is scalable, modular, powerful, affordable, and easy to use. The grant management system has tracking reports which present information in formats designed to meet the needs of a wide range of users

 

The Grant Management System module tracks information such as:

  • Project Status (active or pending)
  • Project Closure Details
  • Grant status (active or pending)
  • Principal investigators as well as their effort
  • Funding agency (MHRD, Government, etc.)
  • Proposal type (New, Competitive renewal, etc.)
  • Award type (Training, Book Purchase, Manpower, etc.)
  • Project periods
  • Fund Allocation Details

 

Key Features:-

 

The key features of this module are:-

 

      • Project Details
      • Institution Management
      • Grant Agency Information
      • Account Head details
      • Grant Period Details
      • Fund Allocation Details
      • Grant Allocation Management
      • Expense Entry
      • User Management
      • Report Generation

 

The Projects Tool mainly used to create new projects and subprojects under the System We can view the list of Projects under the Grant Management System.

The Institution Management Tool is used to create institutions.We can also view the list of Institutions under the project..

Grant Agency Tool will list the list of agencies that provide the Grant for various projects. We can create a new agency also.

Account Head Tool is used for performing the financial transactions of the project. We can create Accounts head and Sub Account Heads.

Grant Period Tool is used for tracking the period allotted for a each grant and the transaction occurred during that particular grant period.

Fund Allocation Tool will provide the details of the fund allocated for the projects including the details such as Project Code, Project Investigator’s Name, Project Start date, Project End date, Grant Agency Name, Amount Allocated etc.

Grant Allocation Managenment The Grant Allocation Management System will give details about the grant allocation of Sub projects, Headwise grant allocation, Grant Received Details etc

 

Expense Entry Tool gives the details of the expenses occurred for various heads.

 

User Management Tool is used for creating a user for the application.

 

Report Generation Tool generates various reports such as Head Wise Expenditure, Institution wise Consolidated Expenditure, Fund Utilization, Grant Summary Report, Grant Agency Report, Project Associate Reports etc.

 

Functionality:-

 

1.  Administration

The basic setting up of Grant management System is done by the administration. Admin has the privilege to create Projects, sub projects, Institution, Participants and Grant Agency. Admin can define the various account heads also. Admin can create the fund set up with the created projects and Grant Agency. Admin can also create new users.

 

 

 

Projects

 

All the existing projects details will be listed in the Project List.

 

 

 

We can create a new project by clicking New Projects button. Enter the project details and click the create button.

 

 

Then new project will be created and added to the projects list.

 

 

We can also create subprojects under one project by clicking the Sub Projects button. Here we can enter the sub project details and click on the create button.

 

 

Then a message  will be shown on the top of the screen “Projects List” and the details will be added to the Sub Projects list.

 

 

 

 

We can set the project status by clicking the Project Closure link. Here we can enter the status and Percentage of Completion of the project. Then click the create button to add the details. The message  will be displayed on the screen and the details will be listed on the bottom of the screen.

 

 

 

Institution

 

All the existing Institutions under the projects will be listed in the Institution List.

 

 

We can create a new institution by clicking the New Institution button in the same screen. Enter the institution details and click the create button. Then the message  will be displayed on the screen institution list and the details will be added to the institution list.

 

 

 

Grant Agency

 

All the existing Grant Agencies will be listed in the Grant Agency List. 

 

 

We can create a new Grant Agency by clicking the New Grant Agency button in the same screen. Enter the grant agency details and click on the create button. Then the message will be displayed on the screen grant agency list and the details will be added to the List.

 

 

 

 

Account Head

 

All the existing Account Heads created will be listed in the Account Head List.

 

 

We can create a new Account Head by clicking the New Account Head button in the same screen. Enter the account head details and click on the create button. Then the message will be displayed on the screen the Account heads list and the details will be added to the list.

 

 

 

 

Sub Account Head can be added for each Account Head by clicking Add Sub Account Head link in the Account Head list. Enter the sub account head details and click on the create button. Then the message will be displayed on the screen account head list and the details will be added to the sub account head list.

 

 

 

 

 

 

Grant Period

Grant Period is used for tracking the period allotted for each grant and the transaction occurred during that particular grant period. We can define the period by clicking the New Grant Period button in the same screen.

 

 

Enter the grant period details and click on the create button. Then the message display on the top of the screen Grant period List and the details will be added to the Grant period list

 

 

 

 

Fund Allocation

 

All the Fund Allocation details can be entered in the Fund Allocation Page. Click on the save button and the details will be added to the list which is shown in the same page.

 

 

 

 

User Management

 

The admin can create a new user. The users will be listed on the screen. New user can be created by clicking the “New User” button.

 

Enter the User details and click the “create” button. The new user details will be added to the User List.

 

 

By clicking the “show” button the details of that particular user will be displayed.

 

 

Click the “edit” button to edit the user details.

 

 

Here we can update or delete the user.

 

 

2.                Funds Management

 

 

All the details of the project such as the Project Name, Fund Allocation details of the project, Fund Utilized, Balance Fund, Project Duration details will be available by clicking  ERP on the main screen.

 

 

After clicking ERP  on the main screen, the following details will be displayed..

 

 

 

Project Sub Allocation

 

The fund allotted for the existing sub projects will be listed here. We can allot funds for new sub projects in the same screen. After entering the fund allocation details of the sub project click the “create” button.

 

 

The details of  subproject allocation are added to the  list  in the same screen and a message New Grant Allocation Created is displayed on the top of the screen.

 

 

By clicking the button “Project List” we can move to the main page.

 

 

We can enter the grant allocation status by clicking  “Grant Withdrawal/Closure” button in the Sub Project Allotment list. Enter the status details and click “save” button.

 

 

The grant allocation status is created and a message  is displayed on the same screen.

 

 

Grant Allotment by Ext Agency

 

Here the details of grant allocation granted by an external agency will be entered. The allocation details of the subprojects will be entered. It includes the Granter, Participant, amount etc. The external agency will be the Granter and the institution will be the Participant After entering the details click “create” button.

 

 

The details of  subproject allocation are added to the  list  in the same screen and a message New Grant Allocation Created is displayed on the top of the screen.

 

 

The grant allocation status can be created by clicking  “Grant Withdrawal/Closure” button in the Sub Project Allotment list. Enter the status details and click “save” button.

 

 

The grant allocation status is created and a message  is displayed on the same screen.

 

 

Head Wise Allocation

Here allocation details of Account Heads  included in the project  such as Grant Allocation, Amount, Grant Period etc. will be entered and click “create“ button.

 

 

The details will be added to the Current Heads list in the same screen and a message   is displayed on the top of the screen.

 

 

Grant Received

The grant received for the project in headwise will be displayed here. Also we can receive the Grant using this screen. For that enter the details of grant received and click “create” button.

 

 

The details of grant received will be added to the ‘Received Grants’ list and the amount received will be added to the ‘Summary of Grants Received’ list which already contains the grant allotment details in headwise. Also a message   is displayed on the screen.

 

 

Expense Management

 

Record Expense

 

We can add new expenses entries such as Date of Expense, Account Head, Expense Amount etc. for the projects in the Expense Entry Screen. Period Wise Expenses and  Account Head Wise Expense Summary details also be displayed  in the screen by selecting the Record Expense link in the main page. Enter expense details and click “create” button.

 

 

The details will be added  to the Period Wise Expenses list and a message  will be displayed on the top of the screen. The expense amount will be added to the Account Head Wise Expense Summary list and also the balance amount  get updated. Inorder to enter the new expense details click “List”.

 

 

 

 

  The Period Wise Expenses list will be cleared and we can select a new period for  entering new expense details.

 

 

 

View Expense for a Period

 

The expenses occurred for a particular period can be viewed here. Enter the period and click “List Expenses”.

 

 

The expense details within the given period will be added to the Current Expenses list which is on the same screen.

 

 

View Expenses Headwise

 

The details of Headwise expenses occurred will be displayed here.

 

 

Report Generation

 

Various forms of  Grant Allocation/Project Status reports can be generated by clicking on the icon Reports in the main page.
