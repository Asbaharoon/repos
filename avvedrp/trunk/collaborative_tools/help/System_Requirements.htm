System requirements:

To configure and set up ' Virtual Lab Collaborative Platform', following system requirements should be met.

 

Server Requirements:

 

  • 1GB RAM

  • 40GB hard disk

  • dual core processor

  • High speed internet connection

 

 

Client Requirements:

 

  • 512 MB RAM

  • 20 GB hard disk

  • Web browser: Microsoft Internet Explorer 7.0 or later, Mozilla Firefox 3.0 or later, Google chrome 7.0,Mac Safari 5.0 or later.

  • Flash player Version 8 or later

  • Internet connection (256 kbps)

  • Intel Core™ Duo 1.83GHz or faster processor
