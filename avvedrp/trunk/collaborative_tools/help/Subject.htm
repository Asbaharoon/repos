Add/Edit Subject:

 

All the subjects that come under a university can be viewed in this page. It also allows the user (privileged roles) to add, edit and delete subjects in a university.

 

 

 

a) Add New Subject:

 

1. Click on the "Add New Subject" below the table of subjects.

2. A small space will appear where you can add the name of the subject and its description.

3.Click "Add" button.

 

 

b) Edit a Subject:

 

1. You can edit a particular subject by clicking on the "edit" icon provided along with each subject.

2. A small space appears where you can edit the appropriate changes.

3. Click on the "Save" button to save the changes.

 

 

c) Delete a subject:

 

1. Click on the “delete" icon provided along with each subject.

2. A pop-up window will appear and you can confirm the deletion by clicking on the "OK" button.

 

 

 

 

 

[Important: All the above mentioned functionalities can only  be performed, if the access provided by the administrator.]
