Manage Role:

The role of each user can be managed in this module.

                1. Select the “Edit” icon of the corresponding user.

                2. A small window appears where you edit the User role.

                3. Click on the “Save” button to confirm the operation.

 

          

Roles and their functionalities:

  1. System administrator : He/She can view and edit the content of all universities.

  2. University admin : He/She has got the privilege to view and edit the contents of all subjects of the respective university only.

  3. HOD : He/She got the privilege to view and edit contents of only those subjects assigned to HOD. The subject is assigned to the HOD as per his/her department by the university admin.

  4. Faculty : The privilege is assigned to the faculty by the university admin .Normally this users has only limited privileges like editing experiment details only.

  5. Reviewer : This user has the privilege to review the experiment created by the other users.

  6. Guest : Guest users will only have read permission from the web site home page.
