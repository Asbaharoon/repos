Installation in Linux Server:

  1. Install LAMP (Linux, Apache, MySQL, PHP) in the hosting server. Make sure the mysql is also installed along with it.

  2. Create a folder in the htdocs folder of LAMP installation.(for example: vlab) (PROJECT_ROOT in config.php)

  3. Check out the files from the following link in to this folder <link> .

  4.  Edit the config.php file.   You can change the default theme (DEFAULT_THEME) to your choice.  Give the PROJECT_ROOT name, the same name of the folder we createdin www (say vlab). Change the PROJECT_EMAILID, with the email id of your project group.

  5. Create a database giving a database name (say: vlab_database). Get the mysql database dump from db_dump folder and execute the same in created database.

  6. Edit the dbmanage.php. Give the dbName as the name created for executing the dump(say: vlab_database)  dbHost =’localhost’.  Update the dbUsername and dbPasswordwith that you provided while installing the mysql. (By default Username will be ‘root’and ‘Password’ will be blank or ‘root’ itself).  

  7. You installation should be complete by now. You can access the  site using browser with the URL http://localhost/{project folder name}.
