Amrita Virtual Lab interface

 

Introduction:

Thank you for choosing Amrita Virtual Lab interface management system.

 

These guidelines will help you from the installation of  the software in your server through to the usage of the system  with its various tools and properties.

 

Overview topics:
 

 Amrita Virtual Lab interface management system has been developed in php, HTML, Javascript and mysql as database.

The system provides the user with the tools to edit and manage the content and interface of the virtual lab.

An administrator can manage different users and can even lock each users with defined subject. Each users will be assigned with a defined role, like university admin, HOD and teacher.

Administrator can define the privileges for each role. For example, an admin can limit the privilege of a role , say teacher with the privilege of only editing experiments and not adding or deleting.

All users who need access to editing tools has to be authenticated with the login. Users can use there existing google username and password to authenticate with this. After authentication, administrator can define the role of the registered users. ie, the admin can change the role of the user to university admin, HOD or teacher.
