Setting up database:

    1. Create a new database and give an appropriate name.

    2. Select the newly created database and execute the database dumb provided.

    3. Change the dbmanage.php  and give the host name in $dbHost , Username in $dbUsername,

        Password in   $dbPassword and dbname in $dbName.

 

  • $dbHost="localhost";

  • $dbUsername="root";

  • $dbPassword="password";

  • $dbName='vlab_management';
