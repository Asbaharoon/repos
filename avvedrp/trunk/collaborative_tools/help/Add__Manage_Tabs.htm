Add/ Manage Tabs:

You can add, edit ,re-arrange tabs,delete tabs and change the type of the tab in the provided window .

 

 

a) Add a Tab:

 

     1.  Click on the “+” tab  to add a new tab.

     2.  Add option will appear where you can add a new tab.

     3.  Enter the "Tab Name".

     4.  Select the data type of the tab, whether it is a "link" or "text".

     5.  Click on the "Tab Icon", if you want to change the icon of the tab and click on the desired icon from the given icons.

     6.  Click on the "Save" button to confirm the operation.

 

 

 

b) Edit Tab Name:

 

   1. Click on the particular tab to edit its tab name.

   2. Edit the tab name in the respective text field.

   3. Click on the "Edit" button. (If tab name given is same as already existing tab , when you click on the edit button , it will show an error message "Sorry same tab already exist in this experiment, try different name".)

 

 

[Important: This operation is applicable only on the added tabs, not on the default tabs (Theory, Procedure, Self -Evaluation, Assignment, Reference, Feedback) . If you still want to edit the default tabs, then it has to be deleted and re-add  the tab.]

 

 

 

c) Changing the data type of tab:

 

  1. Click on the particular tab for which data type has to be changed.

  2. Select the data type of tab from the given options such as Q-N, Link, and Text.

  3.  Click on the “Change” button to confirm the operation.

 

 

 

 d) Delete Tab:

 

1. Click on the particular tab you want to delete.

2. Click on the “Delete” button to delete the tab.

3. A pop-up window will appear to confirm the operation.

4. Click "OK" to continue.
