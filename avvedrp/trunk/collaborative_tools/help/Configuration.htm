Configuration:

After installation,all the below mentioned config variables can be modified on config.php as per the project needs.

 

//Choose the default theme name.

 define('DEFAULT_THEME','amrita');

 

//Name of the folder that we create in root directory to copy the files. (www/vladmin)

define('PROJECT_ROOT','vladmin');

 

//Provide the email id of the project. All emails will be send from this email id.

define('PROJECT_EMAILID','virtual_labs@amrita.edu');

 

//Name your project.

define('PROJECT_NAME','Amrita Virtual Lab');

 

//choose the language file from the lan folder

define('LANG','en_in');
