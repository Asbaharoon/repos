Edit Experiment Details:

 

You can also edit the details of the experiment by clicking on either the "Edit Experiment Details" icon on the left side of the page or click on the experiment itself.

 

1. Select Subject ,Topic ,Experiment that you would like to edit.

2. You can edit tab name/delete tab and change the type of the tab in the provided window.

3. Clicking on the respective tab like Theory, Procedure, Self -Evaluation, Assignment, Reference, Feedback etc to change its content using the text editor provided.

 

 

 

 

Equation editor

 

4. After editing the respective changes, click on the "Submit" button.
