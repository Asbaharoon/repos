Faculty View Visualization

Dashboard

Login using a faculty username and password will be redirecting to a faculty dash board which contains following views

  1. LMS usage of the all the courses handled by the facult. By clicking on any of the course will show LMS usage of the all the modules in that course+average grade of the components of that course
  2. Year wise LMS usage of that particular course also seen

Detailed View (Visualization Charts)

By clicking Home link of the faculty dashboard will redirect the user to a new course selection page

  • User can select any of the course to see its detailed view
  • Click Ok
  • This will direct to the Dash board of the selected course

Detailed View - Course Dashboard

There are two categories of informations are availabe

  • Student Performance
  • Course Analytics

Student Performance

Providing following menu options

  1. Visual Details : Displays a Visualization chart based on the module selected by the user. This will show module usage across the all the students enrolled in the course
  2. Date Utilization : Displays the course usage of the selected date across the students
  3. Summary : Selected Students Course usage across the modules of the course

Course Analytics
