Dashboard

Admin user: This is the admin privileged user of the university. He can view following visualizations from his dash board itself. The dashboard provides following options.

  • LMS usage across the Institutes registered under the university
  • Faculy-Student LMS usage of each institutes.
  • Selected Course's LMS usage across the institutes
  • Selected year wise LMS usge across the institutes.

Detailed View

Click Home link from the Menu. It will enables the detailed visualization view for the university admin.

1. Select Institute Name from the combo box

2. Select the year

3. Click ok. This display detailed view of the University admin

Detailed View of University admin

Comprise two options

1. Course analytics : Tracking and visualizing course wise statistics across the student

2. Student Performance : Tracking and Visualizing Student activity across a series of online courses containing various resources.

Course analytics

Displays following options

1. Summary . Displays Summary of all the courses including module usage

2. Visit Details : Displays total LMS usage across the courses

3. ADHOC tools : Providing a drill down reporting facility in course analytics
