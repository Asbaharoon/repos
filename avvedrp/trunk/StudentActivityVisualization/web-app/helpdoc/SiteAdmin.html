Registration

University Registration

A University level admin user can register into DIVE

  • The details saved for this profile are username,password, university name and address
  • This site admin is the key representative of the university and uses the DIVE on behalf of the university.

Profile Activation

  • On completing the details capture for creating the university admin, a confirmation email with the activation link and login details will be sent to the email address that was entered during registration.  
  • Account will get activated by clicking the activation link provided in the email.

There after , the particular user can log in as university admin for the newly created institution.

Institute Admin/Institute Registration

A Institute Admin user can register into DIVE by providing following details

  • Username, password, Univeristy name(to be selected), Institute Name, Institute Address
  • There is a provision to add multiple LMS for an institute.
  • On completing the details capture for creating the institute admin, a confirmation email with the activation link and login details will be sent to the email address that was entered during registration.

Faculty/Student Registration

Any faculty/student user can register into the system using the user registration module. They have to supply following details

  • username,password, University name(to be selected from list), Institute Name(to be selected from the list),LMS name(to be selected from the list).
  • If they are selecting any LMS from the list, They have to supply the username which they are using in the selected LMS
