Dashboard

Institute Admin's Dashboard contains the visualization charts based on the institute's LMS usage. The following are the main visualization views of Institute's Admin

  1. Faculty-student LMS usage of the institute : By default It is showing LMS usage of the Faculty and student. That is the aggregate activity count from faculty and student
  2. course Wise LMS usage : This will shows the LMS usage across the courses in that insitute. Clicking on any of the course will show the average grade across the components of that selected course.

Data Transformation

This will enable the institute admin to transform its LMS data into DIVE's database. The following steps to be followed for this functionality.

  1. Register LMS
    1. You have to enter the database details of your LMS
    2. Pressing Update button will save these details
  2. Transform will extract the data from the LMS and store these data into the DIVEs database
