Overview topics
 

 

 Amrita Virtual Lab interface management system has been developed in php, HTML, Javascript and mysql as database. This tool provides the user an interface to add, edit and delete Subjects, topics under each subject, subtopics and experiments.

An administrator can manage different users and can even lock each users with defined subject. Each users will be assigned with a defined role, like university admin, HOD and teacher.

Administrator can define the privileges for each role. For example, an admin can limit the privilege of a role say teacher with the privilege of only editing experiments and not adding or deleting.

All users who need access to editing part of the tools need to be authenticated with the login. Users can use there existing google username and password to authenticate with this. After authentication, administrator can define the role of the registered users. ie the admin can change the role of the user to university admin, HOD or teacher.
