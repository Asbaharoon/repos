Edit Experiment:

 

You can view the list of the experiment that comes under a particular topic of a subject in this page. It is also accessible by clicking on the experiment name directly.

 


a) Add New Experiment:

 

     1. Click on the label: "Add New Experiment” provided below the table of experiments.

     2. A Small window  will appear where you can add the name, description and the type of the experiment such as Simulator, Animation, Remote trigger etc.

     3. Click on the "Add New" button to confirm the addition.

 (Once you add a new experiment, you will be redirected to "Edit Experiment Details" page for the further editing of the added experiment.)



    
 b) Edit the experiment name and description:

 

  1. Click on the "Edit" icon provided along with each experiment to edit an experiment.

  2. A small window will appear where you can edit the name, description of an experiment.

  3. Click on the "Save" button to confirm the edit operation.

 

 

c) Delete experiment:

 

1. Click on the “Delete" icon provided along with each experiment to delete the particular experiment.

2. A pop-up window will appear to confirm the delete operation.

3. Click "OK" to confirm the operation.

 

 

 

 

[Important: All the above mentioned functionalities can only  be performed, if the access provided by the administrator.]
