HELP For You
Entering Search Criteria

There are four main text boxes in OPAC that provides you with various options of searching the Union Catalogue.

1. Search by Author, Title, Subject:

For more specific searches, you can look for an author, title or subject by keying into the respective boxes. To search by author, for example, key in the author's name underthe "Author" text box and click on the Search button to retrieve the list of library collections that you are looking for.

2. Search by Keyword:

Quick and easy way to search. Recommended for anyone not sure of a library collection's title, author or subject. This feature will return a list of collections where author, title or subject contains the word you've keyed in.

3. Using OR:

To use the operator OR, simple key in your search separated by OR in the keyword box. For example, keying in "Bastin OR Computer" in the keyword box will return a list contain either articles written by Bastin or subject Computer. Alternatively, you can key in "Bastin OR Leong" in the author field to find works done by either authors.

4. Using AND:

There are 2 ways to use the operator AND : - Key in "Bastin AND Borneo" in the keyword box and a list containing works by Bastin on Borneo will be returned. - Key two or more of the boxes for author, title and subject and a list with author and title and subject will be returned. For example, key in "Bastin" in author box and "Borneo" in title box will return a list by Bastin AND with title Borneo.
