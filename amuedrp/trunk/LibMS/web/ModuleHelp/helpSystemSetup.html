6 System Setup

From this Module you can customize LibMs according to your library, you can set how much fine should be charged to the member and the period of relax of fine,you can set lebel/hierarchy of the library,sublibrary,departmental library,seminar library, book banks etc,you can add and publish notices for the members of the library,you can add faculty and manage them, you can add departments and courses run by various departments,you can define member types such as students, employee,sraff etc of the library.
