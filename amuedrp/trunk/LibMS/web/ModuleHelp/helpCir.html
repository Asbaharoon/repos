Register

The librarian/Administrator of the library can Register Member of the user by clicking Register Link.





Enter Member ID of the user to register. The Member Id can be enrollment number or any other Id number belonging to the member.





Fill all the fields and click Submit button





If registration is done Record Inserted Successfully message will appear.





You can also Create account here or you can Skip this step to create account later.





Enter Card Id and Click Submit button





If account is created successfully a confirmation mail regarding username and password is sent to the user
