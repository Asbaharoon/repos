OPAC

Multilingual OPAC



By using OPAc we can search records in different languages(English,Hindi,Urdu,Arabic).OPAC keeps records of the documents in four different languages to make Searching easier. We can Manage Search Records by by using somekeywords.Search can be done via Simple Search,Browse Search,Additional Search,Advance serach,ISSN/ISBN search,Callno search,Accession Search.
By using OPAC we can do-

Narrow your search by Advance or Additional Search link option.
Use Browse Search to look through entire database alphabatically.
Check new books, series etc. using New Arrivals menu given in left.
Check journals using journals menu given in left.
Check Your Account history using MyAccount Option.
Use Help to get out of any trouble you faced during any procedure.
Use Exit to get out of OPAC.

Browse Search

 Browse search is used to Filter record on the basis of author,title,publisher alphabatically (eg. when you enter author name and click find you will get list of record matching that author name alphabatically).





Additional Search

 Additional search is used to filter record on the basis of author,title,subject etc.This is used to Narrow your search by Advance or Additional Search.







Advance Search

 Advance Search is used to filter record on the basis of author,title,isbn etc.Other fields are mention in this page







ISBN/ISSN Search

 his search is used to filter record on the basis of ISBN/ISSN.





Call No Search

 This is used to filter record on the basis of call no.





Accession No.Search

 Accession no is the unique identity of the book(eg two books of the same title and author name can have even same call no but different accession no.).Every book book in the library will have different accession no.Accession no will give you all the information of the particular book.











New Arrivals

 In this module user can see new books/serials/journals/magezines arrived in the library.



Member Registration

 A User can request for registration in this module.If librarian or Admin permits him he will get user name and password through email.The user will login through user name nad password and can manage his account.This account will give the user information about his fine and pending request for a book.





My Account

 A libary user can login through this module to know detail of his account.if the user is not registered in this module he can register himself through Member Registration module.





Notices
All the notices published in system setup module are published here.this is a public plateform for all users.The libraria can publish notice here through system setup module(System setup->Manage Notice->Add Notice).





Locations
In this module user can see varios locations like Reseach Division/Textual Studies Division etc. Librarian or admin can add locations through System Setup module on the main page.



Feedback

 In this module user can give comments and suggestions to make library facility better.
